กิจกรรม

  เป็นสมาชิกคณะกรรมการสภานักเรียน ปีการศึกษา 2563 ตลอดระยะเวลาที่ดำรงตำแหน่งกระผมได้ปฎิบัติหน้าที่ อย่างสุดความสามารถ ผมได้ช่วยจัดกิจกรรมต่างๆของ โรงเรียน ไม่ว่าจะเป็น กิจกรรมนันทนาการหรือกิจกรรม จิตอาสา

  กระผมได้เป็นนักศึกษาวิชาทหาร ได้เข้าร่วมกิจกรรมจิตอาสาต่างๆ เช่น การทำความสะอาด โรงเรียน ชุมชน วัด และ กระผมได้รับการอบรมในเรื่อง การจราจรและได้รับหน้าที่ให้เป็นนักศึกษาวิชาทหารจราจร ค่อยดูแล การจราจรบริเวณ หน้าโรงรถของโรงเรียน

ผลงาน

  ได้รับรางวัลเหรียญทอง ในการแข่งขันการสร้าง Webpage ประเภท Web Editor ระดับชั้น ม.1-ม.3 งานศิลปหัตถกรรมนักเรียน ระดับภาคเหนือ ครั้งที่ 66

  ได้รับรางวัลเหรียญทอง ในการแข่งขันการสร้าง Webpage ประเภท Web Editor ระดับชั้น ม.1-ม.3 งานศิลปหัตถกรรมนักเรียน ระดับภาคเหนือ ครั้งที่ 67

  จากการแข่งขันทำให้ผมรู้สึกชื่นชอบ ในการทำเว็บไซต์มากขึ้น และยังได้รับประสบการณ์จากการแข่งขันมาพัฒนาเว็บไซต์ของตัวเองให้ดียิ่งขึ้น

  ได้รับรางวัลเหรียญทอง  รองชนะเลิศอันดับที่  2    ในการแข่งขันการตัดต่อ ภาพยนตร์ ระดับชั้น  ม.4-ม.6  งานศิลป-หัตถกรรมนักเรียน   ระดับเขตพื้นที่การ-ศึกษา ครั้งที่ 69

    เป็นเว็บไซต์เศรษฐกิจพอเพียง  เรื่อง โรงเรียนประหยัดพลังงาน เป็นเว็บไซต์ที่จัดทำขึ้นโดยใช้ Google Site และเว็บไซต์นี้ได้จัดทำส่งใน รายวิชา คอมพิวเตอร์
